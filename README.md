# Tic Tac Toe (Java Swing)

A simple, interactive Tic Tac Toe game built using Java Swing.

## Features
- **3x3 Grid**: Classic Tic Tac Toe gameplay.
- **Visual Feedback**: Pink and Light Gray backgrounds for players X and O.
- **Win Detection**: Automatically detects rows, columns, and diagonal wins.
- **Draw Detection**: Detects when the board is full with no winner.
- **Reset Functionality**: Easy-to-use reset button to start a new game.

## How to Run
1. Ensure you have Java (JDK 11 or later) installed.
2. Open a terminal in the project directory.
3. Run the following command:
   ```bash
   java main.java
   ```

## Controls
- Click on any empty cell to place your mark (X or O).
- The game will notify you when a player wins or if it's a draw.
- Click the **Reset** button to clear the board and start over.
